## How to Use

1. Create an empty GitHub repository named `dotnet-azure-devops-reference-architecture`.
2. Copy all files from this pack into the repository root.
3. Open the repo in Codex.
4. Ask Codex to read `AGENTS.md`.
5. Run the prompt in `prompts/00-bootstrap-repository.md`.
6. Continue release by release.
